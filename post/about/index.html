AES加密算法

异或概念

表示法⊕ XOR
含义：相同为0，不同为1
练习：
0 XOR 0 = 1
0 XOR 1 = 1
1 XOR 0 = 1
1 XOR 1 = 1

加密解密最最基本原理

举例：
明文：0101 密钥 1100

加密：
0101
1100
—
1001

解密：
1001
1100
—
0101

一次性密码

即便暴力破解，一次性密码本也无法被破译，即使解密也无法确定解密的是否为原文。

优点

  • 数学上是”无条件完全安全的”

缺点

  • 传输
  • 不能重用
  • 长短问题
  • 保存
  • 不能压缩
  • 每次都要生成密钥

应用：国家间专线
